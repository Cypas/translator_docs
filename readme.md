## 简介

团子翻译器是一款屏幕识别`实时翻译软件`，通过OCR识别屏幕特定范围内的文字，然后将识别到的文字调取各大厂的翻译，并输出翻译结果。

- 官网地址:https://translator.dango.cloud
- 搭载3种ocr识别(本地和两种在线ocr)，满足不同条件下的使用
- 9种翻译源的选择
- 账号系统，能够自动云端保存配置
